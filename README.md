# Diamondnaire - Telegram Slot Game

A full-stack commercial slot game with Telegram login, real payments, and global leaderboard.

## Setup

```bash
npm install
npm start
```

## Deploy

- Frontend: Deploy `public/` folder to Vercel
- Backend: Deploy to Railway/Render

## Features


- Telegram Login
- Real Telegram Payments (Stars)
- Server-side slot game logic
- Global leaderboard
- Ice freeze mechanic
- Multiplier tickets system
